Signo de la Suma

Las reglas que aplicaremos para este ejercicio son las reglas fundamentales de la suma y la resta:

  1. En una suma algebraica de números con el mismo signo se conserva el signo.
  2. Todo número sumado con el cero tiene como valor el mismo número.
  3. Si la suma algebraica es de signos diferentes prevalece el signo del número cuyo valor absoluto sea mayor.
